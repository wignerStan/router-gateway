# Domain Glossary

Generated by BDD Formulation on 2026-03-07.

This glossary defines the ubiquitous language used across all feature files.
Maintain consistency: same concept → same term everywhere.

## Core Terms

| Term | Definition | Used In |
|------|-----------|---------|
| Route | A specific path from request to LLM provider through a credential | Route Planning, Route Execution, Learning & Statistics |
| Route Candidate | A potential route before selection, containing model, provider, and credential info | Route Planning |
| Route Decision | The final selection of primary route and ordered fallbacks | Route Planning, Route Execution |
| Credential | Authentication details for accessing a provider | Route Planning, Health Management |
| Provider | An LLM service vendor (Anthropic, OpenAI, Google, etc.) | All Contexts |
| Model | A specific LLM model identifier | Request Classification, Route Planning |
| Capability | A feature requirement (vision, tools, streaming, thinking) | Request Classification, Route Planning |
| Constraint | A hard filter that eliminates infeasible routes | Route Planning |
| Utility | A score representing route desirability based on features | Route Planning |
| Fallback | An alternative route to try if primary fails | Route Planning, Route Execution |
| Session | A multi-turn conversation identifier | Route Planning |
| Affinity | Preference for maintaining the same provider across a session | Route Planning |
| Health State | Current status of a credential (Healthy, Degraded, Unhealthy) | Health Management |
| Planner Mode | Strategy for route selection (Learned, Heuristic, SafeWeighted, Deterministic) | Health Management |
| Outcome | Result of a route execution (success/failure with metrics) | Route Execution, Learning & Statistics |
| Prior | Inherited statistical baseline for new routes | Learning & Statistics |
| Cold Start | Routing without historical data | Learning & Statistics |
| Token Estimate | Approximate token count for context window sizing | Request Classification |
| Request Classification | Process of extracting requirements from raw requests | Request Classification |
